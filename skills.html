Ruby on Rails

Ruby on Rails is a no-nonsense programming framework. Based off of the Ruby language, Rails is easy to learn and can make any website dynamic and just plain awesome. With hundreds of open-source plugins online, known as rubygems, there is always something for you to include in your site!

Web Fundementals

HTML 5 The foundation of the internet

HTML is a way to show information. It stands for HyperText Markup Language and is the basis for the entire internet. So basically, knowing HTML is a must for every Web Developer. But what does it do? HTML uses a system of tags to mark up content in an easy to view way on a website.

CSS 3 Make sites look cool

CSS is a comprehensive framework that is used to style websites. Without it, this website would just be a white page with black text. Boring right? CSS allows you to color, style, and even animate the content on websites.

Javascript Because static is boring

Javascript allows for interactivity. This client-side language allows for a plethora of actions and events to occur, straight from your web browser. Try it yourself! Click the menu button on the top left corner of this page. See the cool animations? Thanks to Javascript, all of that is possible.
